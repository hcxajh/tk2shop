# tk2shop 项目分析报告

## 1. 项目概述

**项目名称**：tk2shop

**项目类型**：TikTok Shop 商品数据采集与发布工具

**主要功能**：
- 从 TikTok Shop 商品页面自动采集商品信息（标题、价格、SKU、图片等）
- 下载商品相关图片（主图、SKU 图片、描述图片）
- 将采集的数据发布到 Shopify 店铺

**应用场景**：
- 用户提到 TikTok Shop 商品链接时自动采集
- 批量商品采集
- TikTok 商品下载
- 从 TikTok 搬家到 Shopify

## 3. 核心功能分析

### 3.1 商品采集流程

**流程步骤**：
1. **参数解析**：解析 TikTok 商品链接
2. **配置文件选择**：从配置池中选择可用的 AdsPower 配置文件
3. **浏览器操作**：打开 AdsPower 浏览器，获取 CDP URL
4. **页面导航**：导航到 TikTok Shop 商品页面
5. **验证码处理**：检测并等待用户完成验证码
6. **数据提取**：
   - 提取商品标题（使用 `meta[property="og:title"]`）
   - 提取主图（使用特定的 CSS 选择器）
   - 提取 SKU 变体信息
   - 提取价格和划线价
   - 提取商品描述和描述图片
7. **图片下载**：下载主图、SKU 图片和描述图片
8. **数据保存**：生成 product.json 文件，保存采集结果
9. **浏览器关闭**：根据情况关闭浏览器或断开连接

**关键技术点**：
- 使用 Playwright 进行浏览器自动化
- 使用文件锁机制防止多进程同时使用同一配置文件
- 处理 TikTok 页面的动态加载和验证码
- 优化图片 URL，获取原始分辨率图片

### 3.2 并发采集功能

**功能特点**：
- 支持多个 URL 并行采集
- 支持从文件读取 URL 列表
- 可配置并发数量
- 自动检测和清理过期锁文件
- 启动时自动过滤无效配置文件

**并发安全**：
- 使用文件锁防止多进程抢同一配置文件
- 自动检测锁持有者进程是否存活，死了自动清理

### 3.3 Shopify 发布流程

**流程步骤**：
1. **参数解析**：解析商品目录和目标店铺
2. **配置加载**：加载店铺配置，找到目标店铺
3. **数据读取**：读取采集的商品数据
4. **浏览器操作**：打开 AdsPower 浏览器，连接到 Shopify 后台
5. **页面导航**：导航到 Shopify 新建商品页面
6. **信息填写**：
   - 填写商品标题
   - 填写价格和划线价
   - 上传商品图片
   - 设置 Product Options（如果有多个 SKU）
   - 填写每个变体的价格和 SKU
   - 填写商品描述
7. **保存商品**：点击保存按钮，完成发布

**关键技术点**：
- 使用 Playwright 模拟人工操作 Shopify 后台
- 支持从 SKU 名称自动提取选项信息
- 处理 Shopify 页面的动态加载
- 支持批量图片上传

## 4. 配置管理

### 4.1 AdsPower 配置（profile-pool.json）

```json
{
  "description": "tk2shop 采集配置表",
  "updated": "2026-03-23",
  "outputDir": "/root/.openclaw/TKdown",
  "profiles": [
    1896325
  ]
}
```

**配置项说明**：
- `outputDir`：采集结果输出目录
- `profiles`：AdsPower 配置文件 ID 列表

```json
{
  "description": "tk2shop 店铺配置表",
  "updated": "2026-03-24",
  "stores": [
    {
      "storeId": "vellin1122",
      "name": "vellin1122",

  ]
}
```

**配置项说明**：
- `storeId`：店铺 ID
- `name`：店铺名称
- `profileNo`：对应的 AdsPower 配置文件 ID

## 8. 应用场景与使用方法

### 8.1 单个商品采集

```bash
node scripts/collector/runner.js '{"tiktokUrl": "https://shop.tiktok.com/us/pdp/商品slug/商品ID"}'
```

或通过环境变量：

```bash
TK_TIKTOK_URL="https://shop.tiktok.com/..." node scripts/collector/runner.js
```

### 8.2 并发采集

```bash
# 多个 URL 并发
node scripts/collector/spawn.js "url1" "url2" "url3" --parallel 3

# 从文件读取 URL 列表
node scripts/collector/spawn.js --file urls.txt --parallel 5
```

### 8.3 发布到 Shopify

```bash
node scripts/publisher/upload-product.js 2026-03-23/001 --store "店铺名称或profileNo"
```

## 11. 总结与建议

### 11.1 项目总结

tk2shop 是一个功能完善的 TikTok Shop 商品采集与发布工具，通过自动化浏览器操作，实现了从 TikTok Shop 采集商品数据并发布到 Shopify 的全流程。项目采用模块化设计，具有良好的可维护性和扩展性。

2. **支持更多电商平台**：扩展支持其他电商平台，如 Amazon、eBay 等
3. **增加数据清洗功能**：对采集的数据进行自动清洗和标准化
4. **优化图片处理**：增加图片压缩和格式转换功能
5. **添加监控和报警**：增加任务执行状态监控和异常报警
6. **提供用户界面**：开发简单的 web 界面，方便非技术用户使用
7. **增加 API 接口**：提供 RESTful API，方便与其他系统集成
8. **优化并发性能**：进一步优化并发采集的性能和稳定性

### 11.3 技术展望

随着电商平台的不断发展和变化，tk2shop 项目可以继续演进，适应新的平台特性和技术趋势。通过持续改进和扩展，该项目有望成为电商数据迁移和自动化运营的重要工具，为电商从业者提供更高效、更可靠的解决方案。
